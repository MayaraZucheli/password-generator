# Password Generator (Python)

A simple and secure password generator built in Python.
This script allows users to create strong, customizable passwords based on selected character types.

---

## Features

✅ Choose password length
✅ Include lowercase letters
✅ Include uppercase letters
✅ Include numbers
✅ Include symbols
✅ Ensures at least one character from each selected type
✅ Password strength indicator
✅ Cryptographically secure generation using `secrets` module

---

## Technologies Used

* Python 3
* `secrets` (secure random generation)
* `string` (character sets)

---

## How to Run

### Clone the repository

```bash
git clone https://github.com/MayaraZucheli/password-generator.git
```

### Navigate to the folder

```bash
cd password-generator
```

### Run the script

```bash
python password.py
```

## Example

```
How many characters should your password have? 12
Include lowercase letters? (y/n) y
Include uppercase letters? (y/n) y
Include symbols? (y/n) y
Include numbers? (y/n) y

Generated password: A$7kLm2!Qp9@
Strong password
```

---

## Password Strength Rules

* **Strong** → 12+ characters and symbols included
* **Medium** → at least 8 characters
* **Weak** → less than 8 characters
